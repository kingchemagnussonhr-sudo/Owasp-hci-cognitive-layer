# OWASP Cognitive Vulnerability Companion Framework
### Human–Centered Security Layer for LLM Applications

This repository contains a companion framework to the OWASP Top 10 for LLM Applications, focusing on cognitive vulnerabilities (CV-01 Automation Bias, CV-02 Confirmation Bias) and a three-layer Human–Computer Interaction (HCI) security architecture.

For full background, see the markdown documents in this repository.
